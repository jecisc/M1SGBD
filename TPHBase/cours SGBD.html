TP HBase

HBase est une base de données orientée colonnes, construite au dessus du système de fichier HDFS.

1 Le shell

Pour lancer le shell, tapez dans une console la commande hbase. Vous êtes maintenant connecté à la base. La commande quit vous permettra de vous déconnecter et quitter ce shell.

La commande help vous donnera une aide sur les principales commandes du shell. La commande help "COMMAND" vous donnera de l'aide sur la command COMMAND en particulier.

Cette page présente un tableau synthétique des commandes shell de HBase, n'hésitez pas à vous y référer.

Vous allez essayer quelques commandes de shell, en prenant comme exemple la création d'une liste de contacts.

1.1 Création de la table

Votre table sera nécessairement dans l'espace de nom correspondant à votre login linux. La commande create prend en paramètre le nom de la table puis les noms des familles de colonnes.

Exemple :
create 'caronc:maTable', 'fam1', 'fam2'
crée une table 'caronc:maTable' avec deux familles de colonnes, 'fam1' et 'fam2'

1.3 Ajout d'une ligne

On ajoute une cellule à la fois, à l'aide de la commande put.

Exemple :
put 'caronc:maTable', 'JD', 'fam1:prenom', 'Jean'
ajoute dans la table 'caronc:maTable' une cellule pour la ligne de clé 'JD', dans la colonne 'prenom' de la famille 'fam1'.

put 'caronc:maTable', 'JD', 'fam1:nom', 'Dujardin'
ajoute dans la table 'caronc:maTable' une cellule pour la ligne de clé 'JD', dans la colonne 'nom' de la famille 'fam1'.

1.4 Requête : obtention d'une ligne

On utilise la commande get pour obtenir une ou plusieurs valeur(s) associées à une clé.

Exemple :
get 'caronc:maTable', 'JD' 
permet d'obtenir toute les valeurs des colonnes de la ligne ayant pour clé "JD"

1.5 Modification

On utilise également la commande put, car lorsqu'on ajoute une valeur pour une cellule qui existe déjà (même ligne, même colonne), on crée une nouvelle version de cette cellule.

1.4 Requête : obtention de plusieurs lignes

Pour lire le contenu d'une table, on utilise la commande scan.

Exemples :
scan 'caronc:maTable'
Donne toutes les valeurs de la table 'caronc:maTable'

scan 'caronc:maTable', {COLUMNS => 'fam1'} 
Donne les valeurs des colonnes de la famille 'fam1' de la table 'caronc:maTable'

Le shell permet aussi d'appliquer des filtres pour une sélection sur les valeurs.

Exemples :
scan "caronc:maTable",{FILTER => "ValueFilter(=,'substring:je')"}

ou plus précisemment,
scan "caronc:maTable",{FILTER => "SingleColumnValueFilter('fam1','prenom',=,'substring:je')"}

Question

1.7 Suppression d'une table

Ce paragraphe est là juste pour information, ne supprimez pas votre table elle va vous servir par la suite.

Pour supprimer une table, il faut la désactiver, puis la supprimer.

Exemple :
disable "caronc:maTable"
drop "caronc:maTable"

On peut aussi vider le contenu d'une table sans la supprimer, avec la commande truncate. en fait, cette commande désactive la table, la supprime et la recrée pour éviter de garder la trace des lignes supprimées.
